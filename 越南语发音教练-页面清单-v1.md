## 4. 页面 01：启动页 / 落地卖点页

### 页面目标

- 承接广告投流流量
- 在最短时间内说明产品卖点
- 推动用户进入发音体系选择与免费试练

### 页面定位

- 这是转化型落地页，不是普通首页
- 第一屏最重要的是让用户理解“这是一款专门纠正越南语发音的工具”

### 必须模块

1. 顶部标题区
   - 主标题：专注越南语发音纠正
   - 副标题：北越 / 南越双体系，跟读打分，错误定位

2. 核心卖点区
   - 发音纠正更准
   - 北越 / 南越分体系训练
   - 零基础也能练
   - 录音后快速出分

3. 体验说明区
   - 先选体系
   - 免费试练
   - 看完整结果
   - 满意后联系客服购买完整版

4. CTA 按钮区
   - 主按钮：立即体验
   - 次按钮：了解完整版

### 建议信息层级

- 第一屏只保留 1 个主卖点：发音纠正更准
- 其他卖点作为辅助，不要平均铺开

### 关键交互

- 点击“立即体验” -> 发音体系选择页
- 点击“了解完整版” -> 可滚动查看下方卖点，或跳联系客服页

### 页面状态

- 默认展示状态
- 网络异常状态

### 原型备注

- 不建议在这一页要求登录
- 不建议在这一页放太多课程内容介绍

---

## 5. 页面 02：发音体系选择页

### 页面目标

- 让用户明确选择北越或南越
- 为后续试练与学习路径绑定发音体系

### 必须模块

1. 页面标题
   - 选择你的发音体系

2. 说明区
   - 北越和南越是两套不同发音标准
   - 后续示范音和纠音标准将跟随所选体系

3. 体系卡片区
   - 北越发音卡片
   - 南越发音卡片

4. CTA 区
   - 按钮：开始免费试练

### 卡片内容建议

- 名称
- 简短描述
- 一个试听按钮
- 被选中状态

### 关键交互

- 选择北越 -> 进入北越免费试练
- 选择南越 -> 进入南越免费试练

### 页面状态

- 未选择状态：按钮不可点
- 已选择状态：按钮可点

### 原型备注

- 这里建议加入“试听标准音”入口，帮助用户做第一感知

---

## 6. 页面 03：免费试练页

### 页面目标

- 让用户真实体验核心纠音能力
- 完成“听示范 -> 录音 -> 提交评分”闭环

### 必须模块

1. 页面头部
   - 当前体系标识：北越 / 南越
   - 当前为免费试练提示

2. 练习内容区
   - 越南语原文
   - 中文释义
   - 中文近似发音提示

3. 标准音频区
   - 播放标准音按钮
   - 可重复播放

4. 录音区
   - 开始录音
   - 结束录音
   - 重新录音

5. 提交区
   - 提交评分按钮

### 关键交互

- 点击播放标准音
- 开始录音 / 结束录音
- 提交评分 -> 免费试练结果页

### 页面状态

- 未播放标准音状态
- 已播放状态
- 未录音状态
- 已录音待提交状态
- 提交中状态
- 提交失败状态

### 原型备注

- 页面要尽量简单，不要让零基础用户分心
- 免费试练优先使用零基础最容易感知价值的内容

---

## 7. 页面 04：免费试练结果页

### 页面目标

- 用完整结果证明产品价值
- 推动用户联系客服并进入成交链路

### 必须模块

1. 结果总览区
   - 总分
   - 是否达标

2. 三项分区
   - 完整度
   - 准确度
   - 流利度

3. 错误定位区
   - 原文高亮错误位置
   - 错误列表说明

4. 对比播放区
   - 再听标准音
   - 回放自己的录音

5. 转化区
   - 联系客服
   - 登录保存结果
   - 继续体验更多内容（如有）

### 关键交互

- 点击“联系客服” -> 联系客服页
- 点击“登录保存结果” -> 登录授权页
- 点击“再听一遍” -> 标准音播放
- 点击“听我的录音” -> 回放用户录音

### 页面状态

- 达标状态
- 未达标状态
- 音频回放失败状态

### 原型备注

- 免费试练结果不能做得太弱，否则转化理由不够
- 这一页是核心转化页

## 22. 设计注意事项

1. 第一版最重要的不是做复杂视觉，而是把纠音主链路做清楚。
2. “听标准音、录音、看结果、重练”必须是最短路径。
3. 落地页与结果页承担强转化职责，信息层级必须清楚。
4. 练习页与结果页承担核心产品价值，不能被无关信息打断。
5. 北越 / 南越体系标识要在关键页面持续可见，避免用户混淆。
6. 达标 / 未达标状态、已通过 / 未通过状态必须视觉上明显区分。
